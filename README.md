# ts-react-parcel-sass
Starter [TypeScript + React + Sass + Parcel]

## Getting started:
- Install Parceljs if you haven't already: `npm i -g parcel-bundler`
- Install dependencies: `npm i`
- Run: `parcel index.html`

## TSlint:
```
npm i -g tslint typescript
```
